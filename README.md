# 📄 AI Resume Review Assistant

An AI-powered ATS Resume Analyzer built with Streamlit that helps job seekers evaluate and improve their resumes based on job descriptions.

## 🚀 Features

* Upload Resume (PDF)
* ATS Keyword Matching
* Resume Structure Analysis
* Resume Strength Analysis
* Professional ATS Score
* AI Resume Review
* Skill Recommendations
* PDF ATS Report Download
* Job Description Comparison

## 📊 ATS Analysis Includes

* Resume Match Score
* Structure Score
* Strength Score
* Missing Keywords Detection
* Matched Skills Detection
* ATS Compatibility Feedback

## 🤖 AI Features

* Resume Review using Gemini AI
* Resume Improvement Suggestions
* Personalized Skill Recommendations

## 🛠️ Tech Stack

* Google Gemini AI
* PyPDF2

```bash
git clone https://github.com/GhaziaAslam/AI-Resume-Review-Assistant.git

cd AI-Resume-Review-Assistant

pip install -r requirements.txt

streamlit run app.py
```
