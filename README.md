# AI Resume Review Assistant

An AI-powered ATS Resume Analyzer built with Streamlit and Gemini AI.

## Features

* Resume PDF Upload
* ATS Keyword Matching
* Resume Structure Analysis
* Resume Strength Analysis
* Professional ATS Score
* AI Resume Review
* Skill Recommendations
* PDF Report Download

## Tech Stack

* Python
* Streamlit
* Gemini AI
* ReportLab
* PyPDF

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
